PÁGINAS NUEVAS
🛒 /pos — Caja (dashboard + POS)
Vista principal del día: total vendido, cantidad de ventas, top productos.
Botón "Nueva Venta" → abre modal/drawer con grilla de productos táctil + comanda lateral. Al cobrar: descuenta stock y registra la venta.
🍽️ /products — Lista de productos
Tabla/grilla de productos con filtros por categoría. Cada fila muestra nombre, categoría, variantes con precios, estado. Botón crear + acciones editar/desactivar.
🍽️ /products/:id — Detalle de producto
Tabs: Info (nombre, descripción, categoría, imagen) | Variantes (tabla editable de variantes + precios) | Receta (si has_recipe: lista de insumos + cantidades) | Historial (ventas recientes del producto)
🏷️ /product-categories — Categorías
CRUD simple de categorías. Similar a la página ExpenseCategories existente. Color + ícono configurables.
📦 /stock — Inventario
Vista unificada de stock: productos simples (stock por variante) + insumos de recetas (supply stock). Alertas visuales para ítems bajo mínimo. Ajuste manual de cantidades.
